Node Code

Node Code allows nodes to change their output based on whether or not any particular input or output has a connection

Letting you write JavaScript code is unfortunately too dangerous, so I had to write my own simplified interpreter. That technically makes it an interpreter running on an interpreter

Syntax

The syntax is basically JavaScript, but with a very limited feature set

Variables don't have a type, you can use let/var like JavaScript, or nothing at all like Python.

There are no functions

If statements must have brackets, and can't have an else

The only operators are Add (+), Subtract (-), Multiply (*), Divide (/), Not (!), And (&&), Or (||)

// This code assumes there is a Param1 input on the node
myVar = "Hello";
if (Param1)
{
    myVar = myVar + " " + Param1;
}
myVar = myVar + " World!";
return myVar;

Nubs

To refer to a nub, use it's name as defined on the node

Properties:

  • - isDef
    true or false based on whether the nub is connected to another node.
  • - format
    The name of the nub internally, this would be something like %0
  • - type
    The name of the nub's type, as defined in the node
