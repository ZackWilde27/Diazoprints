# Diazo Prints

It's an online modular node-based code editor with a look-and-feel inspired by Unreal Engine's blueprints

At the moment it supports writing C, C#, JavaScript, PHP, Python, and Structured Text.

Modules can be made to add support for any additional languages, or add libraries for the existing languages

You can use the code editor right now [here](https://zackwilde.ca/diazoprints), hosted on my website. The reason I'm putting the source code on GitHub is below

<br>

## Why on GitHub?

I have realized that a project like this cannot be fully completed on my own, at least if it's open source there's a chance it gets the attention of people who want to add a language or library, and if that happens then maybe one day this will become a viable competitor to code editors like VS Code, or at the very least a hands-on teaching tool for introducing people to programming.

<br>
